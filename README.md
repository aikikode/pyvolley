pyvolley
========

Python clone of famous [blobby volley](http://www.blobby-volley.net/) using [cocos2d](http://cocos2d.org/) for GUI and [pymunk](http://www.pymunk.org) for physics

How to install
--------------
###Install dependencies
####Linux users:
```bash
pip install cocos2d
pip install pymunk
```
###Run the game
####Linux users:
```bash
python ./pyvolley.pyw
```
